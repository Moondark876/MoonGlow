# Syntax (Concept)

> **Basic Operators**

- *Subtraction Operator*: `-`
- *Addition Operator*: `+`
- *Multiplication Operator*: `*`
- *Float Division Operator*: `/`
- *Integer Division Operator*: `//`
- *Modulus Operator*: `%`
- *Assignment Operator*: `=`

**Sample Source Code**

``` length = 98 - 23```
